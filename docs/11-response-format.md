统一响应格式与错误码

1. 通用成功响应

```
{  "code": 0,  "message": "ok",  "data": {},  "request_id": "uuid" } 
```

1. 通用失败响应

```
{  "code": 40001,  "message": "参数错误",  "data": null,  "request_id": "uuid" } 
```

1. 错误码表（先定最小集）
   - 0：成功
   - 40001：参数错误
   - 40101：未登录或 token 无效
   - 40301：无权限
   - 40401：资源不存在
   - 40901：状态冲突（如重复点赞、重复关注）
   - 42201：业务校验失败（如作品不可见）
   - 42901：请求过于频繁
   - 50001：服务器内部错误
   - 50301：AI 推理服务不可用
